## 🚀 Installation Instructions

### Prerequisites
- XAMPP installed
- Git installed
- GitHub account

### Setup Steps
1. Clone repository into `C:\xampp\htdocs\`
2. Create database `alumni_portal`
3. Configure `config/database.php`
4. Start Apache & MySQL in XAMPP
5. Visit `http://localhost/alumni-portal`
